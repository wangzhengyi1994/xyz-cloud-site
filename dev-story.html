01 — INPUT MATERIALS

项目开始前，我拿到了什么

XYZ Cloud 是一个 AI 云计算平台的海外营销官网。设计师和产品经理提供了以下规范和素材。

01

Figma 设计令牌

完整品牌色系——蓝色 10 级梯度（主色 #4362ff）、灰色 11 级、白色透明度 10 级。间距 4px→120px，全部导出 CSS 变量。

02

字体规范

UI 字体 JetBrains Mono 全站等宽，代码字体 Courier Prime。12 档字号，字重仅 300 和 500。

03

OBJ 3D 模型

5 个 OBJ 格式 3D 模型，原始 10-20MB / 7-15 万顶点，需预处理降采样到 5000 点导出 JSON。

04

点阵 SVG

1915×924px 的 dot-matrix.svg（543KB），数千个 <rect> 元素。

05

页面结构与文案

12 个区块——Hero、核心价值、AI 平台 5Tab、品牌故事、隧道、地球、CTA、Footer。双语 EN/CN。

06

技术约束

必须 Three.js r86（不可升级），单文件 index.html，无构建工具。

05 — CHALLENGES & DECISIONS

踩坑与决策

每一个"回退"都是快速实验和理性判断。

Problem

地球上台湾消失

低精度 TopoJSON + 大 step，台湾没有陆地点。step=1 有 65k 点太卡。

Solution

高精度 + 手动补点

countries-50m.json + step=2（~16k）+ 手动补台湾/海南/斯里兰卡。

Problem

点阵 SVG 性能崩溃

数千 rect 用 CSS Transform 缩放，浏览器重排卡顿。

Solution

Canvas + 涟漪算法

DOMParser 提取坐标 → Canvas 逐帧。涟漪 12px/帧 + smoothstep 放大镜。

Problem

Core Value 暗色失败

立方体和文本在暗色背景上对比度不好。

Solution

4 分钟果断回退

提交后立即还原，CLAUDE.md 记录"确认保持白色"。

Problem

Cosmic Spectrum 喧宾夺主

GSAP 全屏滚动光谱完全抢走注意力。

Solution

大胆砍掉

display:none 观察 → 确认不需要 → 彻底删除 + 移除 GSAP CDN。

10 — AI COLLABORATION

人机协作开发模式

这个项目从第一行代码到最终部署，全程由人类开发者与 Claude AI 协作完成。CLAUDE.md 是"AI 的规范手册"。

AI 负责

代码生成与算法实现

所有 Three.js 特效代码、Canvas 2D 矩阵运算、TopoJSON 地球渲染、粒子变形插值算法——由 Claude 生成初始实现。

11 — LESSONS LEARNED

复盘与收获

一个设计师，用 AI 把脑子里的特效落地成代码。这条路上踩过的坑和想明白的事。

01

设计师最大的痛：看得到，做不出

脑子里清楚想要 5000 粒子变形、地球飞线、数据隧道的效果，Figma 里也画得出概念稿——但打开编辑器就卡住了。Three.js 文档看不懂，矩阵运算写不来。AI 之前，这些特效只能等开发排期，等到的还不一定是你想要的。

02

AI 让"不好看，再改"成为可能

以前改一个特效参数要提需求、排期、联调。现在是：看一眼→"粒子太稀了"→2 分钟后新版本出来→"颜色偏了"→再 2 分钟。数据隧道 8 轮微调、点阵动画 7 次迭代——这种密度的打磨，以前根本不可能。

03

审美判断是人不可替代的价值

AI 能写出 Three.js 代码，但它不知道"这个动画看着不舒服"。Core Value 暗色背景 4 分钟回退、Cosmic Spectrum 果断砍掉——这些决策只有设计师能做。代码可以生成，品味不行。

04

不需要懂代码，但必须懂体验

不懂代码怎么实现，但知道怎么做出来效果好、性能好。高级审美和交互直觉才是驱动 AI 输出高质量结果的真正引擎——你要能精准描述"哪里不对"，AI 才知道往哪改。

05

2 天做完不是重点，一个人做完才是

以前这种项目：设计 → 开发 → 联调 → 改需求 → 返工，最少两周。现在设计师自己闭环——从 Figma 令牌到 Three.js 粒子，从构思到上线，中间没有任何沟通损耗。

FINAL RESULT

从一张空白页到
9 个视觉特效的云平台官网

96 次提交，2 天核心开发，4000 行代码，1.3MB 总体积。
